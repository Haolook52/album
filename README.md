# ImageShare - 图片分享平台

一个功能完整的现代图片分享网站，支持图片上传、展示、分类、搜索等核心功能，采用响应式设计，可在不同设备上提供良好的用户体验。

## 🚀 功能特性

### 核心功能
- **图片上传**: 支持拖拽上传、批量处理、文件格式验证
- **图片展示**: 网格/列表视图切换、图片详情页
- **智能搜索**: 基于标题、描述和标签的全文搜索
- **分类管理**: 多级分类系统，支持自定义分类
- **响应式设计**: 完美适配桌面、平板和移动设备

### 用户体验
- **懒加载**: 图片懒加载技术，提升页面性能
- **图片优化**: 自动生成不同尺寸的图片版本
- **快速搜索**: 实时搜索建议和筛选功能
- **社交互动**: 点赞、浏览统计功能

### 管理功能
- **后台管理**: 完整的统计仪表板和内容管理
- **数据分析**: 浏览量、点赞数、热门分类等统计
- **批量操作**: 支持批量上传和管理

## 🛠️ 技术栈

### 前端技术
- **React 18** + TypeScript - 现代化前端框架
- **Tailwind CSS** - 实用优先的CSS框架
- **React Router** - 客户端路由管理
- **Zustand** - 轻量级状态管理
- **Vite** - 快速构建工具

### 后端技术
- **Node.js** + Express - 服务器端框架
- **SQLite** - 轻量级数据库（开发环境）
- **Multer** - 文件上传中间件
- **CORS** - 跨域资源共享

### 部署方案
- **GitHub Pages** - 静态页面部署
- **Vercel** - 现代化部署平台
- **自定义域名** - 支持绑定自有域名

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
# 启动前端开发服务器
npm run dev

# 启动后端服务器（需要另一个终端）
npm run server
```

### 构建生产版本
```bash
npm run build
```

### 类型检查
```bash
npm run typecheck
```

### 代码检查
```bash
npm run lint
```

## 🌐 部署指南

### GitHub Pages 部署
1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择部署分支为 `gh-pages`
4. 访问 `https://<用户名>.github.io/<仓库名>`

### Vercel 部署
1. 连接GitHub仓库到Vercel
2. 自动检测并部署
3. 配置环境变量（如果需要）
4. 访问自动生成的域名

### 自定义域名配置
1. 在域名服务商添加CNAME记录
2. 在部署平台配置自定义域名
3. 等待DNS生效
